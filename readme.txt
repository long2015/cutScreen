软件名称：截图小工具 cutScreen v1.0
软件说明: 
	程序分三部分：显示类 widget、截图区域保存类RCutScreen、输入类InputDialog
    widget复制界面显示，RCutScreen保存截图区数据，InputDialog获取输入的width height

使用说明: 按住鼠标左键拖动选择保存区域。
按键说明：
     s     保存区域截图
     f     全屏截图
     q     退出程序
